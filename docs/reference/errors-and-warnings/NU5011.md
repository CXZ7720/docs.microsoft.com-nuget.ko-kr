# <a name="nuget-error-nu5011"></a>NuGet 오류 NU5011

### <a name="issue"></a>문제

어셈블리 또는 프로젝트 파일에서 메타 데이터를 추출 하는 동안 런타임 예외가 발생 했습니다.


### <a name="solution"></a>솔루션

문제를 만드세요 https://github.com/NuGet/Home/issues합니다.
