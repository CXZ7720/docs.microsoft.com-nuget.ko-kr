# <a name="nuget-error-nu5009"></a>NuGet 오류 NU5009
<pre>Cannot find version of msbuild.</pre>

### <a name="issue"></a>문제

NuGet pack 작업 msbuild를 찾을 수 없습니다.


### <a name="solution"></a>솔루션

전달 하세요는 `-MsBuildPath` nuget pack 명령입니다.
