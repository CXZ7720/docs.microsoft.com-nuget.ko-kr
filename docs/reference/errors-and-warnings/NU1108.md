# <a name="nuget-error-nu1108"></a>NuGet 오류 NU1108

<pre>Cycle detected: A -> B -> A</pre>

### <a name="issue"></a>문제
순환 종속성이 발견 되었습니다.

### <a name="solution"></a>솔루션
패키지를 제대로 작성 버그를 해결 하려면 패키지 소유자에 게 문의 합니다.

### <a name="note"></a>참고
Visual Studio 2017의 초기 버전 수 보고이 경고 (NU1606)로.
