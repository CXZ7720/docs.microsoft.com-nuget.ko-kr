<span data-ttu-id="8fa01-105">NuGet은 다양한 위치에서 대상 프레임워크 참조를 사용하여 패키지의 프레임워크 종속 구성 요소를 구체적으로 식별하고 격리합니다.</span><span class="sxs-lookup"><span data-stu-id="8fa01-105">NuGet uses target framework references in a variety of places to specifically identify and isolate framework-dependent components of a package:</span></span>
